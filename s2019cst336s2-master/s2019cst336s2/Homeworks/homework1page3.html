C++ continues to be used in all types of tech. These are 5 ways it is applied in modern times.

  • High throughput, low latency systems like equity trading systems is an area where there is active C++ development.
  • C++ is used extensively in Google including in their search engine code, for deep learning algorithms. Go is used alongside C++.
  • A lot of cloud infrastructure code is in C++ - for example system management software on storage appliances.
  • Products such as VMware ESXi use C++ heavily.
  • NoSQL databases like MongoDB and Tarantool are written in modern C++.
